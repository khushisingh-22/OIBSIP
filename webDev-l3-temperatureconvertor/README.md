# 🌡 Temperature Converter

A simple and interactive **Temperature Converter** built using **HTML5, CSS3, and Vanilla JavaScript**. This web application allows users to convert temperature values between **Celsius (°C), Fahrenheit (°F), and Kelvin (K)** with proper input validation and user-friendly error handling.

## 🚀 Features

* Convert temperatures between Celsius, Fahrenheit, and Kelvin
* Numeric input validation
* User-friendly error messages for invalid input
* Absolute zero validation
* Displays all converted temperature values simultaneously
* Clean and responsive user interface
* Built using pure HTML, CSS, and JavaScript (no external libraries)

## 🛠️ Tech Stack

* HTML5
* CSS3
* JavaScript (Vanilla)

## ▶️ How to Run

1. Download or clone the repository.
2. Open the project folder.
3. Double-click **index.html** or open it in any modern web browser.
4. Enter a temperature value.
5. Select the input unit.
6. Click the **Convert** button to view the converted temperatures.

## 📌 Input Validation

* Accepts only numeric values.
* Displays an error message for empty or invalid input.
* Prevents temperatures below absolute zero:

  * Celsius: **-273.15°C**
  * Fahrenheit: **-459.67°F**
  * Kelvin: **0 K**

## 📸 Output

The application displays the converted temperature in:

* Celsius (°C)
* Fahrenheit (°F)
* Kelvin (K)
