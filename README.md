🚗 DR WASHIT - Doorstep Car Wash Landing Page
📌 Project Overview

DR WASHIT is a responsive landing page developed for a doorstep car wash service. The website allows users to explore the company's services, pricing, and features, and book a car wash directly through WhatsApp.

This project was created using HTML, CSS, and JavaScript with a clean, modern, and mobile-friendly design.

🎯 Objective

The objective of this project is to provide an attractive and user-friendly landing page for a car wash business where customers can quickly learn about the services and book appointments online.
